# PHP Bootstrap Template

## WTF is this?

This is a very simple, super simple PHP template with the following things already 
in place so you don't have to bother to include or configure them:

* Twitter Bootstrap 3.3.5
* Font Awesome 3.2.1
* Google analytics
* Google Webmaster
* A very simple layout with a top menu
* A couple of pages

## How to use?

* Fork it
* Fill variables.inc
* Enjoy

## Why should I use that?

* You need to make a one~three pages app and you are looking for the fastest way to id it...responsive.

## How can I contribute?

* Open an issue asking for stuff
* The maximum allowed size for this template project is 5MB.
